UrJTAG installation on Windows

Method 1(Portable Version)#

  1. Download the zip files: urjtag_windows.zip and driver.zip
    UrJTAG Installation Files (Windows)
    Krypton Drivers
  2. Extraxt/Unzip the file: urjtag_windows.zip and driver.zip
  3. Move to the directory/folder: urjtag_windows/
  4. Run the file: jtag.exe
note
  • If you are using Krypton for the first time install Krypton drivers; steps are mentioned below
  • Right click and run as administrator if normal mode does not work

Steps to Install Device Driver#

  1. Connect the Krypton board to your PC and Power On, you will see notification “USB Device not recognized”.
  2. Open “Device Manager”.
  3. If you see yellow sign for devices under Other devices as shown in the figure below then continue to step-4 else skip to step-9.
  4. Right click on first device (i.e. Dual RS232-HS) and click Update Drivers.
  5. Click on Browser my computer for drivers and then click Browse.
  6. Browse to the folder where driver.zip has been extracted; select Krypton driver_CDM20817 folder and click install.
  7. Repeat steps 4-6 for second Dual RS232-HS device.
  8. Now you can start using urJTAG
  9. Drop down “USB Serial Bus Controller” , Select “Unknown USB Device” , Right click and then click on “Update driver”.
  10. Select “Search Automatically for Updated Driver Software”.
  11. Remove the Krypton USB Cable and reconnect it, If yellow sign disappears, you are good to go, else follow next step.
  12. Drop down “USB Serial Bus Controller” , Select “Unknown USB Device” , Right click and then click on “Uninstall device” , confirm and click on “OK/Uninstall”. Remove the Krypton USB Cable and reconnect it, If yellow sign disappears, you are good to go.
