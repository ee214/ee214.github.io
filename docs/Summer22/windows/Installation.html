Installation

  1. Go to Quartus download page

  2. Make sure you have following options selected on download page
    Edition: Lite
    Release: 20.1.1
    Operating System: Windows

    • If you have fast internet
    • Slow Internet
    Select combined files and click download button.
    Select Individual files and download following components
    1. Quartus Prime
    2. ModelSim-Intel FPGA Edition
    3. MAX II, MAX V device support

  1. Now you will be asked to login. If you already have an account on intel website, use that credentials to login; otherwise register for a new individual account.

    note

    You can use your gmail ID for new account registration.

  2. Once the file has been downloaded

    • If you have downloaded combined files, then extact the downloaded .tar file and run setup.bat file
    • If you have downloaded individual files, then make sure all downloaded file are within same directory and run QuartusLiteSetup-20.1.1.720-windows.exe file

  3. Follow the installation instruction and make sure to include support for MAX V devices and ModelSim.
