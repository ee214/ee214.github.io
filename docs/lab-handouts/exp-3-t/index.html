Combinational Circuit 3

Wadhwani Electronics Lab, IIT Bombay
Tuesday 7th September, 2021

Use structural modelling for this experiment; means instantiate components and use port map to connect those components

Multiplier#

(i) Design [3 Marks]#

Design a multiplier circuit with one 4-bit input and one 3-bit input.

(ii) VHDL description [5 Marks]#

Describe your designed circuit in VHDL.

(iii) Simulation [5 Marks]#

Simulate your design using the generic testbench to confirm the correctness of your description.

note

To do this, use the tracefile given below and modify the testbench given to you appropriately.

(iv) Scanchain [5 Marks]#

Test the correctness of your design using Scanchain
