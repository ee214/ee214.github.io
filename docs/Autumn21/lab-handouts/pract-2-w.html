Serial Comparator​

Write VHDL Description of serial comparator using Behavioural modelling.

Assume that the circuit is in an initial state at time instant mmm. Then for any n≥mn ≥ mn≥m, the output sequences {l(k)}k=mn\{l(k)\}^n_{k=m}{l(k)}k=mn​ and {g(k)}k=mn\{g(k)\}^n_{k=m}{g(k)}k=mn​ are related to the input sequences {a(k)}k=mn\{a(k)\}^n_{k=m}{a(k)}k=mn​ and {b(k)}k=mn\{b(k)\}^n_{k=m}{b(k)}k=mn​ by the following relations

  • l(n)=1l(n) = 1l(n)=1 if and only if a(n)a(n−1)a(n−2)....a(m)a(n) a(n-1) a(n-2) .... a(m)a(n)a(n−1)a(n−2)....a(m) is strictly less than b(n)b(n−1)b(n−2)....b(m)b(n) b(n-1) b(n-2) .... b(m)b(n)b(n−1)b(n−2)....b(m)
  • g(n)=1g(n) = 1g(n)=1 if and only if a(n)a(n−1)a(n−2)....a(m)a(n) a(n-1) a(n-2) .... a(m)a(n)a(n−1)a(n−2)....a(m) is strictly greater than b(n)b(n−1)b(n−2)....b(m)b(n) b(n-1) b(n-2) .... b(m)b(n)b(n−1)b(n−2)....b(m)

Thus, the outputs can be thought of as providing comparison information about the inputs.

In the state diagram

a>ba > ba>b represent inputs a = '1' & b = '0'
a<ba < ba<b represent inputs a = '0' & b = '1'
a=ba = ba=b represent inputs a = '0/1' & b = '0/1'
{01} represent outputs gt = '0' & lt = "1" means a < b
